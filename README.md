My Journey Into Data

After finishing my studies, I started exploring what direction I wanted to grow into.  
Landing my first “semi‑data role” as a specialist within a retail company changed everything for me.

I became increasingly fascinated by Power BI — the idea that you can turn raw data into something visual, meaningful, and useful for real people.  
A colleague then introduced me to SQL, and that opened an entirely new world.  
I was amazed by how someone could pull exactly the data they needed straight from the cloud with just a few lines of logic.

That moment sparked something.

I began studying through Maven Analytics, building my foundation in SQL, Power BI, and data modeling.  
Now I’m applying these skills inside the company:  
- building small dashboards to support the department  
- automating repetitive tasks  
- improving workflows using Power BI and Power Query  

Every project helps me grow a bit more toward my long‑term goal:  
developing into a Data Engineering role, learning more about coding, understanding different languages, and eventually building powerful in‑company tools from scratch.


On this GitHub you’ll find projects where I combine Power BI, SQL, Excel (Power Query) and general data engineering fundamentals (really willing to learn but still a long way to go)
